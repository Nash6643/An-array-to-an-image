# Random Image Generator

This Python script generates a random image using NumPy and the Pillow (PIL) library. The image is saved as a JPEG file and can be displayed.

## Features
- Generates a random RGB image of customizable size.
- Saves the image as a `.jpg` file.
- Displays the generated image.
- Includes error handling for robustness.

## Requirements
Ensure you have the following dependencies installed:

```bash
pip install numpy pillow
